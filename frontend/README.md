# Frontend - Task Platform (JavaScript)

Scaffolded with Vite (React). Key libraries:
- react / react-dom
- react-router-dom
- redux toolkit + react-redux
- axios
- recharts

Scripts:
- npm run dev (starts Vite)
- npm run build

Notes:
- Set VITE_API_URL in `.env` to point to backend (e.g., http://localhost:5000/api)
- Further pages and slices to implement: Tasks, Task Detail, Create/Edit, Profile, Analytics
